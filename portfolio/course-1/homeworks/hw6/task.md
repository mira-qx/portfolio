Задание 1

Создайте класс Movie, который описывает фильм.
Что нужно сделать:
Добавьте три свойства:
Title — название фильма
Genre — жанр
Rating — рейтинг (от 0 до 10)
Добавьте три конструктора:
без параметров (устанавливает значения по умолчанию:
 "Без названия", "Неизвестен", 0);
с одним параметром — title;
с тремя параметрами — title, genre, rating.
Используйте this(...), чтобы более короткие конструкторы вызывали полный.
Добавьте метод PrintInfo(), который выводит информацию о фильме:
Название: ..., жанр: ..., рейтинг: ...

В Main() создайте несколько фильмов разными способами:
var m1 = new Movie();
var m2 = new Movie("Матрица");
var m3 = new Movie("Начало", "Фантастика", 9);

Вызовите PrintInfo() для всех фильмов.

Задание 2

Создайте базовый класс Device, в котором есть:
свойство Name;
метод TurnOn(), который выводит:
 "Устройство включено.".


Создайте два дочерних класса:

Kettle — добавьте метод Boil(), который выводит "Чайник кипятит воду.";
Toaster — добавьте метод Toast(), который выводит "Тостер поджаривает хлеб.".


Пример вызова:

var kettle = new Kettle();
kettle.Name = "Redmond";
kettle.TurnOn();
kettle.Boil();

var toaster = new Toaster();
toaster.Name = "Philips";
toaster.TurnOn();
toaster.Toast();

Задание 3

Продолжение задания 2

В базовом классе Device добавьте виртуальный метод:
public virtual void Beep()
{
    Console.WriteLine("Устройство подаёт сигнал.");
}


В классах Kettle и Toaster переопределите (override) этот метод:
Kettle выводит: "Чайник пикнул: пи-пи!";
Toaster выводит: "Тостер пикнул: динь!".


Пример:
var k = new Kettle();
var t = new Toaster();

k.Beep();
t.Beep();
