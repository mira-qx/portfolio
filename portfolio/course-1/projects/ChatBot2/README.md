# ChatBot — Telegram-бот на ASP.NET Core с интеграцией OpenRouter

## Описание проекта
Telegram-бот, который принимает сообщения пользователя через webhook, сохраняет историю диалога, отправляет контекст в OpenRouter (OpenAI API) и возвращает ответ пользователю. Бот поддерживает контекстные диалоги и несколько полезных команд.

## Функциональность

### Основные команды
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и краткая инструкция |
| `/help` | Список доступных команд |
| `/stats` | Статистика чата (количество сообщений и токенов) |
| `/clear` | Очистить историю диалога |
| `/summarize` | Краткий пересказ текущего диалога |
| `/undo` | Удалить последнюю пару сообщений |
| `/history N` | Показать последние N сообщенийъ |
